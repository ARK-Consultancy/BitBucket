Installation Steps

Step 1:

Install App

Go to Apps > Update Apps List > Update > Search > Automatic Bank Synchronization > Install

Update Apps List is only visible in developer mode. To get into developer mode go to Settings > Developer Tools (Section) > Activate Developer Mode

Step 2:

Create Plaid Account

  1. Go to https://plaid.com
  2. Click the Get API keys button
  3. Fill out form > Create account

If you do not want to create your own Plaid account you can use ours for a flat monthly fee email plaid@lunel.co

Step 3:

Enter Plaid Account Credentials into Odoo

Go to Accounting > Configuration > Settings > Plaid Configuration Settings

Here you will be able to enter in your Plaid Client ID, Public Key, Secret, and select if you are in Sandbox, Development or Production

Step 4:

Configure Plaid Bank Account

Go to Accounting > Configuration > Journals > Select Bank Journal

Next enable the Plaid Bank Synchronization. Once you have enable Plaid on the Journal you will see a new section called Plaid Bank Synchronization.

Here you can select your Bank Statement Creation. This is for how often you want Odoo to create you a new bank statement. You can still import in your transactions each day. And Odoo will put the new transaction into your existing bank statement.

Next click the Configure button

Step 5:

Login to Bank Account

Search for you bank in the Plaid pop-up.

Sign into your bank account and select which accounts you want to import into Odoo.

Step 6:

Configure Bank Account to Journal

Once you are back to Odoo you need to link your Plaid bank accounts to an Odoo Journal.

In the Actions column select and existing Odoo Journal or select Create new journal if you do not already have the journal created in Odoo.

In the Bank Statement Creation column select how often you want Odoo to create a new bank statement.

Next set the Fetch Statement From. This will be the date Odoo will import transactions from. You can set this back up to 24 months. Some banks only allow 12 months.

Remember the Bank Statement Creation is for how often you want Odoo to create you a new bank statement. You can still import in your transactions each day. And Odoo will put the new transaction into your existing bank statement.
